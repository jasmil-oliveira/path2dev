	Páginas Dinâmica (Server side scripts)

	O conhecimento acima se misturará com qq tecnologia voltada a internet, tal como:
	.aspx (asp .net), .jsp (Java Server Pages), .php, .do Java2EE, .py PYTHON, .pl PERL etc...

	No caso do PHP a iniciação necessita de um ambiente de desenvolvimento e testes, preparado.

	O que é esse ambiente?

	O ambiente de infra-estrutura de desenvolvimento de sistema em php, contempla basicamente:

	a) softwares de servidor

	Apache - web server;
	MySql - servidor de banco de dados;
	Php - linguagem de script server-side;

	b) integrated development environment (IDE) (ferramenta de desenvolvimento),( escolha um, faça o download e instale-o )

	VisualCode, phpStorm, Netbeans dentre outras.
